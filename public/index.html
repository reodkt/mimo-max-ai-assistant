AI assistant for all, just an experiment.

Product goal

Build a reliable Indonesian-language productivity assistant for founders, creators, and UMKM teams that need fast summarization, drafting, and planning.

Responsible rollout

API keys stay server-side, input length is capped, unsafe requests are guided by the system prompt, and usage can be rate-limited before launch.

Initial milestones

Week 1: connect credentials. Week 2: test Indonesian prompts. Week 3: add feedback. Week 4: release a limited beta.
